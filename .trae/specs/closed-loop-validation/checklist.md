## 综合验收标准（来自原 spec 的"需实际运行验证"项）

### 稳定性（原 4 项）
- [ ] 连续运行 24 小时无崩溃或内存泄漏 → **B4 压力测试验证**
- [ ] 后台标签页场景下所有功能正常 → **B4 并发测试验证**
- [ ] 快速连续操作不导致竞态条件 → **B4 高频操作测试验证**
- [ ] 异常输入不会导致应用卡死 → **B1 冒烟测试异常场景**

### 性能（原 4 项）
- [ ] 补丁应用后启动时间增加 <2 秒 → **B2 冷/热启动基准对比**
- [ ] AI 响应延迟增加 <100ms → **B2 AI面板响应时间测量**
- [ ] 内存占用增加 <50MB → **B2 内存基线 + B4 泄漏检测**
- [ ] agent-browser 操作响应时间 <500ms → **A2 CDP 延迟采集**

### 兼容性（原 4 项）
- [ ] Windows 10/11 不同版本可用 → **A1 启动兼容性测试**
- [ ] 不同屏幕分辨率/DPI 下 UI 正常 → **B3 回归测试多分辨率**
- [ ] 中文路径和文件名处理正确 → **B1 冒烟测试中文路径场景**
- [ ] 杀毒软件不误报 → **A1 启动过程观察**

### 边界条件（原 4 项）
- [ ] 极长对话（100+ 轮）不导致性能下降 → **B4 长时间运行+模拟**
- [ ] 大文件操作（>100MB）不导致卡死 → **B1 冒烟测试大文件场景**
- [ ] 并发多个 AI 任务不冲突 → **B4 并发多任务测试**
- [ ] 网络断开重连后功能恢复 → **A1 保活重连机制验证**

### 文档补充（原 2 项）
- [ ] README 文档完整 → **C4 报告含使用说明**
- [ ] Schema 文件补充 → **A4 报告含配置说明**
